# Text Generation & Named Entity Recognition (NER)

It demonstrates how to perform **text generation** using GPT-2 and **Named Entity Recognition (NER)** using a BERT model from the Hugging Face Transformers library.

---

## Technologies Used
- Python 3  
- [Hugging Face Transformers](https://huggingface.co/transformers)  
- BERT & GPT-2 models  
- Google Colab
